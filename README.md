1️⃣ What is JSX, and why is it used?
Ans: JSX হলো JavaScript এর একটি syntax extension, যা দেখতে HTML এর মতো।
এটি React এ UI তৈরি করার জন্য ব্যবহার করা হয়।

2️⃣ What is the difference between State and Props?
Ans: State = নিজের ডাটা,Props = বাইরের ডাটা

3️⃣ What is the useState hook, and how does it work?
Ans: useState হলো একটি React Hook যা component এ state যোগ করতে সাহায্য করে।
